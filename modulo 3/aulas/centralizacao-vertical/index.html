para centralizar horizontalmente o conteúdo, era utilizado o margin auto

para centralizar verticalmente e horizontalmente o conteúdo é feita com o position

o position: absolute tem que estar setado no elemento que vai ficar centralizado

o position: relative fica no local aonde o elemento vai ser centralizado

o comando "top" mexe a caixa verticalmente, o comando "left" mexe a caixa horizontalmente

o position considera o vértice superior esquerdo do elemento, não seu centro, para que centralize corretamente, utliza-se o comando "transform: translate(-50%, -50%)

valor positivo movimentará a caixa para direita e para baixo, ja o valor negativo movimentará a caixa para esquerda e para cima
