Git e Github

Git:Um gerenciador de versões, aonde é feito o versionamento do seu código, ou seja, aonde vai ser guardado as versões do seu código dentro do seu computador. (repositório local)

Github:Aonde vai ser hospedada na nuvem as suas versões do projeto, para que consiga acessá-lo de outros computadores ou até outras pessoas consigam ter acesso ao seu código, também torna muito mais prático o trabalho em equipe (cada um com seu repositório local, porém o mesmo repositório remoto). (repositório remoto)

(repositório remoto: "push")

Como Hospedar O Conteúdo

crie um repositório local no local drive do github, aonde você colocará o nome da pasta que quer hospedar e aonde ela se encontra, coloque uma descrição (opcional), escolha a licença (MIT é a geralmente usada)

Logo em seguida o repositório local estará criado, você terá a opção de hospedá-lo no repositório público na opção "Publish Respository", fazendo assim com que o seu repositório seja salvo na nuvem aonde pode ser acessado por você caso queira fazer o backup ou por outros computadores e outras pessoas
