Texto que adapta ao tamanho da tela

Consiste em usar "vw" no lugar de "em" (nao é a melhor maneira de se adaptar, ja que a melhor maneira é o media query, que ainda nao foi ensinado no curso)

Utilizar o "box-shadow inset" para criar uma sombra interna no elemento

Hierarquia de tabelas (simples)

table: abre a tabela

tr: linha da tabela

table header: cabecalho da tabela

td: dado da tabela

CSS de tabela

border-collapse: junta as bordas, para que nao fique aquele vao entre uma borda e outra.

Alinhamento de tabela

text-align: centraliza o texto horizontalmente

vertical-align: centraliza o texto verticalmente

como usar o vertical-align:

  • top
  • middle
  • bottom

Anatomia da tabela

                    TABLE
                               thead
                                       tbody       
                                tfoot

cada um com seu tr,td e th

Transformando texto em CSS

os comandos "text-transform: uppercase;" transforma todo o texto em letras maiúsculas

e o comando "font-variant: small-caps" torna as outras letras menores que a primeiras, para dar contraste a primeira letra mesmo em um texto em letras maiusculas

utilizar o padding para espaçamento interno entre elementos, por exemplo parágrafos, em vez de utilizar o margin, pois utilizando o padding a movimentação é bem mais facilitada
