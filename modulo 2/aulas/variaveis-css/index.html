Variáveis em CSS

:root, permite criar variáveis em CSS, muito prático para facilitar na escolha das cores

funciona da seguinte maneira:

:root { --cor0: #BB4D04; --cor1: #3200E6; --cor2: #453091; --cor3: #2C283C; }

de '--cor0' até '--cor3' são variáveis que armazenam algo dentro delas

você pode acessar essas variáveis pelo 'var()' por exemplo

não se aplica somente a cores, podendo ser aplicada a qualquer informação que se armazenar.

:root { --cor0: #BB4D04; --cor1: #3200E6; --cor2: #453091; --cor3: #2C283C; --fonte: 'Poppins', sans-seriff; }

é um exemplo de root que guardar também uma fonte

pseudo-elemento "after"
