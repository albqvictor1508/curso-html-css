Responsividade

termo criado para o fato de adequar sua aplicação aos demais tamanhos de telas.

até agora foram utilizados os seguintes comandos:

  • max-width: largura máxima da div ou do html todo
  • min-width: largura mínima da div ou do html todo
  • max-height: altura máxima da div ou do html todo
  • min-height: altura mínima da div ou do html todo

(esses 4 comandos também auxiliam na adaptação do conteúdo na viewport)

  • width 100% na imagem: faz com que a imagem se adeque a viewport(tela)
  • picture: para utilizar mais uma imagem para tamanhos de telas diferentes
  • margin:auto : para manter a div ao centro independentemente do tamanho da viewport

Essas foram as tags responsivas utilizadas até agora
