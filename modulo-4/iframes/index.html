Iframes

Iframes: quadros em linha

(quadros com display inline)

Diversos sites não permitem iframes pois pode ser utilizado para extrair dados de outras pessoas

Configurando iframes

é possivel usar width e height nos iframes

O css é soberano ao que é formatado na tag iframe

Por exemplo, se eu colocar 600px de width na tag iframe e colocar 450px no seletor iframe no CSS, o CSS tem prioridade

(scolling depende do seu navegador)

scolling="auto" : se o conteúdo exceder o tamanho do iframe, a barra de rolagem aparece

scolling="yes" : mesmo que o conteúdo não exceda o tamanho do iframe, a barra de rolagem aparecerá

scolling="no" : mesmo que o conteúdo exceda o tamanho do iframe, não aparecerá a barra de rolagem

frameborder é a borda do iframe, que também pode ser feita com o border do CSS
